O `Modal` é um componente utilizado para exibir um conteúdo em uma sobreposição na tela. Ele permite definir a largura máxima e altura máxima do modal.

### Exemplo de uso do Modal

```jsx
<Modal isOpen={true} maxWidth="35rem" maxHeight="30rem">
  Modal Example
</Modal>
```
