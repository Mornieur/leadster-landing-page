O `Header` é um componente utilizado para exibir o cabeçalho de uma página. Ele exibe um logo, um título principal e um subtítulo.

### Exemplo de uso do Header

```jsx
<Header
  minText="WEBNARS EXCLUSIVOS"
  minTitle="Menos Conversinha"
  maxTitle="Mais Conversão"
/>
```
