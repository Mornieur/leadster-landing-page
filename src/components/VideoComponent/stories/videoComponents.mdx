## VideoComponent

O `VideoComponent` exibe uma lista de vídeos com miniaturas e oferece a opção de reproduzir os vídeos em um modal.

O `VideoComponent` exibe uma lista de vídeos com miniaturas e oferece a opção de reproduzir os vídeos em um modal.

### Exemplo de uso do VideoComponent

```jsx
import { VideoComponent } from './VideoComponent';
import videos from './videos.json';

export const Example = () => (
  <VideoComponent videos={videos} isModalOpen={false} />
);
```
